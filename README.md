# Two-Stage-Stochastic-ALM-Model-in-Julia
A fully worked two‑stage stochastic Asset‑Liability Management (ALM) model implemented in Julia using JuMP and HiGHS.   This project demonstrates how to build a realistic pension‑fund ALM engine with scenario‑dependent recourse, CVaR risk control, and transaction‑cost‑aware rebalancing — all within a linear optimization framework.
